Planejador de Atividades
Um aplicativo web simples para planejar e gerenciar atividades diárias, com funcionalidades de criação, edição e exclusão de tarefas.

Funcionalidades
Adicionar atividades: Insira título, descrição, data e hora para cada atividade.

Editar atividades: Clique no ícone de edição para modificar uma atividade existente.

Excluir atividades: Remova atividades clicando no ícone de lixeira.

Armazenamento local: Todas as atividades são salvas no localStorage do navegador para persistência de dados.

Interface responsiva: Design moderno e adaptável a diferentes tamanhos de tela.

Tecnologias Utilizadas
HTML5

CSS3 (com Tailwind CSS para estilização)

JavaScript Vanilla

Como Usar
Abra o arquivo planejador.html em qualquer navegador moderno.

Preencha os campos:

Título da atividade

Descrição

Data

Hora

Clique em "Salvar Atividade" para adicionar.

Para editar, clique no ícone de lápis (azul).

Para excluir, clique no ícone de lixeira (vermelho).

Estrutura do Código
HTML: Define a estrutura da página com formulário e lista de atividades.

CSS: Utiliza Tailwind CSS para estilização responsiva.

JavaScript: Gerencia a lógica do aplicativo, incluindo:

Manipulação do DOM

Armazenamento local (localStorage)

CRUD de atividades

Personalização
Você pode facilmente personalizar:

Cores: Modifique as classes Tailwind (purple, blue)

Layout: Ajuste as classes de container e espaçamento

Funcionalidades: Estenda o script JavaScript para adicionar novas features

Compatibilidade
Testado em navegadores modernos (Chrome, Firefox, Edge, Safari). Requer JavaScript ativado.
